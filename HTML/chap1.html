2. Difference between html and html 5

HTML5 is the latest version of HTML, which includes new features and improvements over previous versions. Some key differences include:

  • New semantic elements (e.g., <header>, <footer>, <article>)
  • Support for audio and video embedding without plugins
  • Improved form controls and attributes
  • Canvas element for drawing graphics
  • Local storage and offline capabilities

HTML :

  • HTML is a markup language used for creating web pages.
  • It provides the structure and content of a webpage.
  • HTML uses tags to define elements such as headings, paragraphs, links, images, etc.
